
# 🌦️ Weather App

A clean and simple weather application that fetches real-time weather data using a public API. Built as part of my journey learning web development, Git, and GitHub. ⚡

---

## ✨ Features

- 🔍 **Search by City** — Get instant weather updates for any city worldwide
- 🌡️ **Live Temperature** — Displays current temperature in real-time
- ☁️ **Weather Conditions** — Shows sky conditions (sunny, cloudy, rainy, etc.)
- 💧 **Humidity & Wind Speed** — Additional weather details at a glance
- 📱 **Responsive Design** — Works smoothly across desktop and mobile
- 🎨 **Simple, Clean UI** — Minimal design for easy readability

---

## 🛠️ Built With

- 🧱 **HTML5** — Structure
- 🎨 **CSS3** — Styling
- ⚙️ **JavaScript** — Logic & API integration
- 🌐 **Weather API** — Live weather data

---

## 🚀 Getting Started

### 📋 Prerequisites
- A modern web browser (Chrome, Firefox, Edge, etc.)
- Internet connection (for API calls)

### ▶️ Run Locally

1. Clone the repository
   ```bash
   git clone file:///C:/Users/krith/Downloads/Weather%20App.html
   ```
2. Navigate into the project folder
   ```bash
   cd weather-app
   ```
3. Open `index.html` in your browser 🎉
